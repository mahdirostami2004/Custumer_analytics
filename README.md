# Customer Analytics (RFM Analysis)

This project performs customer segmentation using RFM (Recency, Frequency, Monetary) analysis with pandas, matplotlib, and seaborn.

## Dataset
UCI Online Retail dataset: [Download here](https://archive.ics.uci.edu/ml/datasets/online+retail)  
Place the file `online_retail.csv` inside the `data/` folder.
